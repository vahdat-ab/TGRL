Please just copy these four ".jar" files inside the plugin folder of your Eclipse.
In order to have the plugins in your Eclipse IDE, you first need to install Xtext and Acceleo.
These two packages can be installed through the modeling section of your Eclipse repository. 
For more instructions, you can visit their official web sites:
Xtext:  https://www.eclipse.org/Xtext/
Acceleo: https://www.eclipse.org/acceleo/
